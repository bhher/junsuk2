# flow-board 만드는 순서

## 왼쪽 체크리스트

| 순서 | 작업 | 파일 |
|:---:|------|------|
| 1 | 프로젝트 생성 | `npm create vite@latest flow-board -- --template react` |
| 2 | 의존성 설치 | `cd flow-board && npm install` |
| 3 | 루트 HTML | `index.html` |
| 4 | 진입점 | `src/main.jsx` |
| 5 | 전역 스타일 | `src/index.css` |
| 6 | 최상위 컴포넌트 | `src/App.jsx` |
| 7 | 카드 폼 (하위) | `src/components/CardForm.jsx` |
| 8 | 카드 (하위) | `src/components/Card.jsx` |
| 9 | 컬럼 | `src/components/Column.jsx` |
| 10 | 보드 | `src/components/KanbanBoard.jsx` |
| 11 | Vite 설정 | `vite.config.js` |
| 12 | 패키지 설정 | `package.json` |

## 의존 관계

```
App.jsx
  └── KanbanBoard.jsx
        └── Column.jsx
              ├── Card.jsx
              └── CardForm.jsx
```

## 권장 작성 순서 (아래→위)

1. **CardForm** — 단순 폼, 의존성 없음  
2. **Card** — 개별 카드, 드래그 속성  
3. **Column** — Card + CardForm 조합  
4. **KanbanBoard** — Column 배치 + 드래그 로직  
5. **App** — columns, cards 상태 + CRUD 핸들러
